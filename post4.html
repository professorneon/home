🌌 Deep Web nədir? – İnternetin Görünməyən Üzünə Dərin Baxış

Yazan: ISKOPISIX | Tarix: 07.06.2025 | Qovluq: Kibertəhlükəsizlik / Dərin Veb

🌐 Deep Web anlayışı

Deep Web (Dərin Veb) – internetin standart axtarış motorları (Google, Bing və s.) tərəfindən indekslənməyən hissəsidir. Burada yerləşən məzmunlar xüsusi giriş və ya autentifikasiya tələb edir və adi yollarla tapıla bilməz. Buraya akademik verilənlər bazaları, dövlət sənədləri, bank sistemləri, tibb məlumatları, abunəlik arxasında olan saytlardakı məzmun və korporativ daxili sistemlər daxildir.

🧱 Deep Web ≠ Dark Web

Qarışdırılmamalıdır: Deep Web geniş və legitim məzmunu əhatə edən sahədir. Onun kiçik bir hissəsi isə **Dark Web** adlanır – anonimlik və xüsusi alətlər (Tor, I2P) vasitəsilə daxil olunan, tez-tez qeyri-qanuni fəaliyyətlərlə əlaqəli olan qapalı sistemdir. Yəni, bütün Dark Web Deep Web-in içindədir, lakin bütün Deep Web Dark Web deyil.

🔐 Deep Web-ə daxil olan sistemlər

  • Bank hesabları, ödəniş sistemləri və elektron kommersiya panelləri.
  • Universitet və tədqiqat bazaları (JSTOR, IEEE, SpringerLink və s.).
  • Şəxsi e-poçt və istifadəçi profilləri.
  • Dövlət portalları və elektron hökumət sistemləri.
  • Korporativ intranet və daxili sənədlər bazaları.

🛠️ Texniki baxımdan Deep Web

Deep Web-ə daxil olan məlumatlar əsasən aşağıdakı səbəblərə görə indekslənmir:

  • Robots.txt: Saytın indekslənməsini məhdudlaşdırmaq üçün istifadə olunur.
  • Dinamik səhifələr: Form vasitəsilə əldə edilən məlumatlar (məsələn, axtarış nəticələri).
  • Şəxsi giriş: Şifrə və autentifikasiya tələb edən məzmunlar.
  • API ilə qorunan sistemlər: Məlumatların RESTful və ya SOAP servislər vasitəsilə təqdim edildiyi sistemlər.

🧠 Deep Web-in əhəmiyyəti

Deep Web kritik informasiya sistemlərinin qorunması və şəxsi məlumatların təhlükəsiz saxlanılması üçün vacib infrastruktur yaradır. Tədqiqatçılar, jurnalistlər, təhlükəsizlik mütəxəssisləri və dövlət qurumları üçün dəyərli informasiya mənbəyidir.

🔎 Misal: JSTOR axtarış nəticələri

https://www.jstor.org/action/doBasicSearch?Query=Cyber+Security

Bu linkə yalnız autentifikasiya və ya abunəliklə daxil olmaq mümkündür – bu səbəbdən axtarış motorları bu səhifəni indeksləmir.

⚠️ Etik və Hüquqi Mövqelər

Deep Web-ə daxil olmaq ümumiyyətlə qanunidir, lakin buradakı məlumatlara icazəsiz giriş kibercinayətkarlıq sayılır. Məlumatlara daxil olarkən etik və hüquqi çərçivələrə əməl olunmalıdır.

🌐 Deep Web ilə Surface Web fərqi

Qat Adı Təsvir
1 Surface Web (Səthi Veb) Axtarış motorları tərəfindən indekslənmiş, ictimai şəkildə açıq olan veb saytlar (məsələn: Wikipedia, YouTube, CNN)
2 Deep Web (Dərin Veb) İndekslənməmiş, giriş tələb edən, ancaq qanuni olan məlumatlar (akademik bazalar, dövlət portalları və s.)
3 Dark Web (Qaranlıq Veb) Deep Web-in kiçik hissəsi, anonimlik və .onion domenləri ilə işləyən, tez-tez qeyri-qanuni fəaliyyətlərlə əlaqəli olan qat

📚 Deep Web-ə Daxil Olan Məzmun Növləri

  • Akademik Verilənlər Bazaları – JSTOR, IEEE Xplore, SpringerLink
  • Bank Sistemləri – Hesabatlar, əməliyyat tarixçələri
  • Tibbi Məlumat Bazaları – EHR (Electronic Health Records)
  • Dövlət və Hökumət Məlumatları – Vergi məlumatları, məhkəmə sənədləri
  • Abunəliklə Açılan Saytlar – Netflix, Spotify və s.
  • Şirkət İçi Sistemlər – VPN-lə qorunan daxili şəbəkələr

🔐 Niyə Axtarış Motorları Bunları Göstərmir?

Axtarış motorlarının indeksləmə etməsi üçün müəyyən texniki şərtlər tələb olunur:

  • Saytın ümumi girişə açıq olması
  • robots.txt və sitemap icazəsinin olması
  • Statik və ya açıq link strukturları

Amma Deep Web saytları isə:

  • Parolla qorunur
  • Sessiyalara əsaslanır
  • Dinamik kontent yaradır
  • robots.txt ilə indeksləmə qadağası var

🧠 Deep Web Texnologiyaları və Quruluşu

📌 Mühüm texnoloji komponentlər:

  • Login / Auth protokolları – OAuth, SSO, LDAP
  • SQL və NoSQL verilənlər bazaları
  • Session Cookies və JWT ilə autentifikasiya
  • Firewall və ACL-lər – IP və rol əsaslı icazələr

🧬 Entropiya və Gizlilik Təhlükəsizliyi:

  • Sessiya tokenləri: 128–512 bit
  • HTTPS və TLS ilə şifrələmə
  • URL formatı: https://portal.gov.az/profile?id=39821&auth=abc123xyz

⚖️ Deep Web-in Hüquqi Statusu

Deep Web-dəki məlumatlar əsasən qanuni və hüquqi çərçivədə qorunur. Misallar:

  • Şəxsi məlumatların qorunması
  • Əqli mülkiyyət hüququ olan kontent
  • Tənzimlənən tibbi və maliyyə məlumatları

🚫 Dark Web ilə bağlı qısa qeyd

Dark Web əsasən anonimlik üçün yaradılmış Tor (The Onion Router) şəbəkəsində yerləşir. Qeyri-qanuni bazarlar, hack forumlar, qara bazar alqı-satqısı, istismar edilən məlumat bazaları burada yerləşə bilər. Dark Web istifadəçiləri üçün **VPN + Tor** istifadəsi minimum təhlükəsizlik tələbidir.

🛡️ Nəticə

Deep Web, internetin görünməyən, lakin həyatın çox önəmli hissəsini təşkil edən qatıdır. Onun dərk edilməsi müasir texnologiya və kibertəhlükəsizlik anlayışlarının ayrılmaz hissəsidir. İnformasiya təhlükəsizliyi, etik hacking və OSINT ilə məşğul olan şəxslər üçün Deep Web haqqında geniş bilik vacibdir.
